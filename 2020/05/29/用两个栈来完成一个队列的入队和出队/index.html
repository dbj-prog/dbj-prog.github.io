用两个栈来完成一个队列的入队和出队

发表于 2020-05-29 | 分类于 数据结构与算法

题目描述

用两个栈来实现一个队列，完成队列的 Push 和 Pop 操作。

思路

队列的特点是先进先出,栈的特点是后进先出,顺序进栈的数出栈后变逆序,再经过一次入栈和出栈就变回了原来的顺序.

  • 如果两个栈都空,执行pop操作应抛出异常
  • 如果stack2空,执行pop操作应先将stack1中的内容出栈并压入stack2中
